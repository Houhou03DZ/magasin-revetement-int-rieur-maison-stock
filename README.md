# Gestion de stock – Magasin Revêtement

Application Java console pour gérer le stock d’un magasin de revêtement intérieur.

## Technologies
- Java
- MySQL (XAMPP)
- JDBC

## Installation
1. Installer XAMPP et lancer MySQL
2. Créer une base de données `magasin_stock`
3. Importer le fichier `magasin_stock.sql`
4. Importer le projet dans Eclipse
5. Ajouter le driver MySQL Connector/J
6. Lancer `Main.java`
